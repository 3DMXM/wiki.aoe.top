# 主页

**inZOI ModKit** 是一个创意工具，允许用户通过自己的想象力重现inZOI的生动世界。  
构建您自己的空间、角色和故事，将它们带入游戏，并与其他玩家分享您的体验。  
本指南是您踏上这段旅程的第一步。
